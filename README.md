# Site-Descodificadas
## Projeto realizado na Trilha 1 Descodificadas 2023-2
### Os requisitos a serem incluídos são:

#### Logotipo;
#### Menu;
#### Título destacado sobre o assunto do site com imagem do fundo;
#### Trecho de texto com plano de fundo colorido;
#### Botão com link interno ou externo;
#### Imagens com descrições;
#### Vídeo;
#### Mapa de localização;
#### Formulário para coletar cadastro;
#### Rodapé com a frase criado por seu nome.
